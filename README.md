# Forced_Oscillator_ODE_Analytical_vs_Simulink
### Non-Homogeneous Second-Order ODE  
### Analytical Solution vs Simulink Numerical Solution

This repository presents the analytical and numerical solution of the non-homogeneous differential equation
x''(t) + 4x(t) = 2 sin(3t)
with initial conditions:
 x(0) = 5
 x'(0) = -3
